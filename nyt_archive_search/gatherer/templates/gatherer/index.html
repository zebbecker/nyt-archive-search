Welcome to Archive Search.

Search for New York Times articles by keyword and download full CSV datasets without writing any code!

Archive Search gives students and researchers easy access to the New York Times article archive. This tool provides a simple graphical interface for the NYT API, and streamlines the process of data collection and formatting.

This project is pedagogical in nature- dead simple, approachable interfaces take precedence over fine grained control. The ultimate goal of this project is to make an important research method accessible to students and researchers without backgrounds in computer science.

Getting Started

Use the demonstration search mode to quickly get started with a small example search. The demo mode gives you full search functionality, but will only download the first ten results.

The New York Times limits how often you can query their API. Therefore, in order to create larger datasets, you will need to acquire and input your own API key. The process is quite simple- click here to get started. Make sure to save your personal API key somewhere private.

Once you have your own API key, you can download as many results as you desire using the full search mode.
